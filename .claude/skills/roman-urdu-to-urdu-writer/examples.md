# Example Usage

## Example 1
Input: "Haqiqat kahon TU tujy khwab lgta ha"
Output: "حقیقت کہوں تو تجھے خواب لگتا ہے"

## Example 2
Input: "mjy bht pasand ha ye geet"
Output: "مجھے بہت پسند ہے یہ گیت"

## Example 3
Input: "tum se milny ko bht eager hun"
Output: "تم سے ملنے کو بہت ایگر ہوں"

## Example 4
Input: "kal subah milte hain agar ho sake to"
Output: "کل صبح ملتے ہیں اگر ہو سکے تو"

These examples demonstrate how the skill converts Roman Urdu to proper Urdu script while preserving meaning and emotional tone.
